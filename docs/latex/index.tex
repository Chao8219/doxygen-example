For Ubuntu 20.\+04, before installing the Doxygen, some other packages/libraries should be installed first. The following list can be helpful\+:
\begin{DoxyEnumerate}
\item CMake\+: one can directly download the binary {\ttfamily .sh} file from \href{https://cmake.org/download/}{\texttt{ CMake Download}}, and run something like\+: {\ttfamily /bin/sh cmake-\/3.\+22.\+1-\/linux-\/x86\+\_\+64.\+sh}.
\item GNU flex\+: simply run\+: {\ttfamily sudo apt-\/get update} and {\ttfamily sudo apt-\/get install flex}.
\item GNU bison\+: {\ttfamily sudo apt-\/get update} and {\ttfamily sudo apt-\/get install binson}.
\item libiconv\+: follow the guide here\+: \href{https://geeksww.com/tutorials/libraries/libiconv/installation/installing_libiconv_on_ubuntu_linux.php}{\texttt{ Installing libiconv on Ubuntu Linux}}
\end{DoxyEnumerate}

After all that, you can install the Doxygen from source\+:

\DoxyCodeLine{}
